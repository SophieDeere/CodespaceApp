# GitTutorial
Dies ist ein Template-Repository mit einer CodeSpace-Konfiguration für ein **Interaktives Git Tutorial**.

## Los geht's in wenigen einfachen Schritten

### Repository erzeugen & Codespace starten
1. Mit dem Button (oben rechts) **[Use this Template]** > **Create new Repository** legen Sie sich Ihr persönliches Repository auf Basis dieser Vorlage an. Wählen Sie Ihren Benutzernamen als `Owner`, stellen Sie die Sichtbarkeit auf `Private`.
2. In Ihrem Repository: Klicken Sie oben rechts auf den grünen Button **`<> Code`**.
3. Wählen Sie den Reiter **`Codespaces`**.
4. Klicken Sie auf **`Create codespace on main`**. 
*(Tipp: Beim ersten Mal kann das Laden ein bis zwei Minuten dauern, da Ihre Umgebung frisch eingerichtet wird.)*
